Добрый день!

Представляю вам дипломный проект на тему "Система бронирования жилья"

Приложение состояи из 3 частей:
1. База данных, используется PostgreSQL
2. Серверная часть написана с использованием Spring Boot
3. Frontend написан на JavaScript, расположен в каталоге src\main\resources\web_site\

Программа позволяет искать жилье по параметрам: Город, Тип, Рейтинг, Свободные даты, Цены.
Для авторизованных пользователей доступно:
Бронирование, просмотр бронирований, отмена бронирования.
Добавление собственных объектов недвижимости с загрузкой фотографий.
Изменение данных личного кабинета.
